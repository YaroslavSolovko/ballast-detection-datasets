# ML-driven Detection and Reduction of Ballast Information

This repository provides the code and experimental configuration used in the paper:

"ML-driven detection and reduction of ballast information in multi-modal datasets"

## Reproducibility
All preprocessing steps, feature selection thresholds, and model hyperparameters are documented in the provided Jupyter Notebooks.

## Requirements
- Python ≥ 3.9
- See `requirements.txt` for package dependencies

## Datasets
Due to licensing and size constraints, datasets are not included.
Links to the original data sources are provided within the notebooks.

## Notes
Large-scale datasets were processed using batch-based sampling and sparse matrix representations where applicable.
